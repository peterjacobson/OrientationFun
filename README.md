
# OrientationFun

game to onboard new collaborators

## how

### install

```
git clone https://github.com/peterjacobson lukekavanagh/OrientationFun
cd 
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
